FAQ

자주 묻는 질문

AI가 틀린 답(환각)을 주지는 않나요?
추측으로 답하지 않고, 질문을 데이터 조회 질의로 바꿔 실제 데이터를 조회한 결과로 답합니다. 답과 함께 사용한 조회 조건을 보여줘 숫자의 근거를 확인할 수 있고, 조회 전용 검사·의미 검증·오류 자동 보정을 거칩니다.
AI가 우리 데이터를 바꾸거나 지우지는 않나요?
AI가 만드는 질의는 조회(SELECT)만 허용합니다. 추가·수정·삭제 같은 변경 명령은 실행 전 검사에서 차단됩니다.
민감한 데이터는 안전한가요?
비밀번호 등 민감 컬럼은 조회 대상에서 처음부터 제외하고, DB 접속 정보는 암호화해 보관합니다. 내부 오류·원시 에러도 사용자 화면에 그대로 노출하지 않습니다. SyncLife는 ISO 정보보호경영시스템 인증을 보유하고 있습니다.
지금 바로 쓸 수 있는 기능은 무엇인가요?
현재 제공 기능은 자연어 데이터 질의, 멀티턴 대화, 물류 도메인 특화 응답, 답변 근거 제시입니다. 비정형 주문서 변환과 문서·리포트 자동화는 개발 예정(로드맵)입니다.
사용자별 접근 권한 제어는 되나요?
현재는 조회 전용으로 안전하게 동작합니다. 사용자 인증과 테넌트별 데이터 격리, 사용자별 세분화 권한은 외부 공개 전 단계적으로 적용할 예정입니다.
데이터 분석을 잘 모르는 직원도 쓸 수 있나요?
네. 평소 쓰는 말로 질문하면 되고 쿼리 문법은 필요 없습니다. 채팅하듯 묻고 이어서 후속 질문을 할 수 있습니다.
